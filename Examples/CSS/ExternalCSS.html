External CSS

Once you start adding a lot of styles to a web page, they quickly start to become difficult to read

To make this easier to manage, styles can be in a different file and linked to the HTML document

To do this, a HTML LINK tag is put at the top of the page - <link rel="stylesheet" href="path/to/file.css" />

When the browser encounters the LINK tag, it will load the file and apply all of the styles in it
