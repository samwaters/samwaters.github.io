CSS Introduction

CSS (Cascading Style Sheets) applies styles to HTML elements to change how they look and act

Styles can be applied to any HTML Element, and can change pretty much anything about it

CSS has two parts:

  • The selector
  • The rules applied to any elements matching the selector

Selectors

Selectors target HTML elements in the web page to apply styles to.

They can be done in 3 different ways:

  • HTML Tag name, e.g. p or h1 - this will apply a style to every tag that matches
  • By element id - a unique ID can be assigned to HTML elements to apply a style to only that one
  • By class name - a class can also be assigned to HTML elements, to apply a style to any elements with that class

Tag Name

This can be any HTML tag element, e.g. div, h1, p

Element Id

An ID can be added to a HTML element, like <div id="cat"> and selected using a hash and the id - #cat

Class Name

A class can be added to a HTML element, like <div class="cat"> and selected using a period and the class name - .cat

Rules

Rules are applied to elements matched by the selectors, and are placed inside curly brackets

A rule has two parts - the property being changed and what it is being changed to, e.g. color:blue

Multiple rules can be applied at once, separated with semicolons - { color:white; font-size:16px; }

CSS Location

CSS is placed in a <style> tag at the top of the HTML page:

            <style>
                span { color:red; }
                #cat { color:blue; }
                .cat { color:green; }
            </style>

Examples

Selecting by HTML Tag

This text is inside a span tag, and has this rule - span { color:red }

Selecting by ID

This text is inside a span tag with id cat, and has this rule - #cat { color:blue }

Selecting by class

This text is inside a span tag with class cat, and has this rule - .cat { color:green }
