HTML Introduction

This is an example of a HTML page

HTML stands for HyperText Markup Language and describes the structure and content of a web page

It is a simple text document that uses tags to describe different structures. Some examples include:

  • H1, H2, H3, H4, H5 and H6 for headings
  • img for images
  • p for paragraphs
  • ul for a bulleted list
  • li for a bullet point
