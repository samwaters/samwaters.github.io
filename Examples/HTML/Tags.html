HTML Tags

There are many different HTML tags, but they all share the same basic premise

A tag has a few different components:

  • They are denoted by angle brackets: < >
  • The tag name, e.g. p or h1
  • Any attributes, if applicable, that give the tag more details e.g. src="images/cat.jpg"
  • Any content, for example the text in a paragraph
  • The closing tag, which matches the opening tag except for a forward slash, e.g. </p>
  • Some tags are called short tags, and include the opening and closing tag together, e.g. <img />

Some examples would include <p>Hello</p> and <img src="images/cat.jpg" />
