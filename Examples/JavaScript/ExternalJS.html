External JavaScript

Like CSS, adding lots of JavaScript to the page quickly becomes unreadable and unmanageable

Scripts can also be placed in other files and included in the HTML page

To do this, a HTML SCRIPT tag is put at the bottom of the page - <script src="path/to/script.js" ></script>

When the browser encounters the SCRIPT tag, it will load the file and execute the script in it once the page has loaded
